📌 Description:
----------------------------------------------------
This is a simple Hangman word guessing game
developed using Object-Oriented Programming (OOP)
concepts in C++. The game supports:

- Single-player mode (select word by category)
- Multiplayer mode (P1 inputs word, P2 guesses)

Key OOP concepts used:
- Inheritance
- Polymorphism
- Abstraction
- Encapsulation

----------------------------------------------------
▶️ How to Run:
----------------------------------------------------
From terminal or command prompt:
    Hangman.exe       (Windows)

----------------------------------------------------
🕹️ Game Controls:
----------------------------------------------------
- Choose mode: Enter 1 (Single) or 2 (Multiplayer)
- Follow the on-screen instructions
- Input guesses as letters
- Exit: Select option 5 from the main menu
